# lab-tooling

A minimal monorepo using **Nx**, **Angular**, **FastAPI**, and **SQLite**.

## Structure

```
lab-tooling/
├─ apps/
│  ├─ frontend/        Angular app (port 4200)
│  └─ backend/         FastAPI app (port 8000)
├─ libs/
├─ infra/
│  └─ compose/
│     └─ docker-compose.yml
├─ data/
│  └─ sqlite/          SQLite DB file stored here
├─ nx.json
├─ package.json
├─ tsconfig.base.json
├─ .env.example
└─ README.md
```

## Prerequisites

- Node.js 20+
- npm 10+
- Python 3.12+
- pip
- Docker & Docker Compose (for container workflow)

## Install dependencies

```bash
# Node / Angular / Nx dependencies
npm install

# Python / FastAPI dependencies
npx nx run backend:install
# or manually:
cd apps/backend && pip install -r requirements.txt
```

## Run with Nx

### Frontend (Angular — port 4200)

```bash
npx nx serve frontend
# or
npm run start:frontend
```

### Backend (FastAPI — port 8000)

```bash
npx nx serve backend
# or
npm run start:backend
```

### Both in parallel

```bash
npm run start:all
```

## Initialize the database

```bash
npx nx run backend:init-db
```

This creates the SQLite tables on first run. The backend also auto-initialises on startup.

## Run with Docker Compose

```bash
# Copy and edit environment variables
cp .env.example .env

# Start all services
npm run docker:up
# or
docker compose -f infra/compose/docker-compose.yml up --build

# Stop all services
npm run docker:down
```

## API endpoints

| Method | Path          | Description           |
|--------|---------------|-----------------------|
| GET    | /health       | Returns `{"status":"ok"}` |
| GET    | /api/items    | List all items        |
| POST   | /api/items    | Create an item        |

### Create item payload

```json
{
  "name": "Example item",
  "description": "optional text"
}
```

## SQLite file location

| Context   | Path                  |
|-----------|-----------------------|
| Host      | `./data/sqlite/app.db` |
| Container | `/data/app.db`        |

The file is created automatically when the backend starts.

## Assumptions

- SQLite is used as a file database — no separate DB container.
- CORS is configured to allow `http://localhost:4200`.
- SSR is included in the Angular app (generated by Nx) but the dev server (`nx serve`) serves the app without SSR in development mode.
